RANDOMIZATION SYSTEM

Whenever a user starts a new exam attempt:

* The system must randomly shuffle questions from the **200-question databank**
* Each attempt must load **a different random set of questions**
* The same user should rarely see the same question twice in a row
* Randomization must occur on both **subject selection and question order**
